# CSE110: Course Selection Management System

This is a **Course Selection Management System** developed for **CSE110: Object-Oriented Programming (East West University)** using **Java Swing**.  

Students can select courses, and management can add or remove courses. The system demonstrates core **OOP concepts** and uses Swing components like **JFrame, JButton, and JTable**.

---

## Files Included

1. **CSE110 Project-ReportAndCode.pdf**  
   - Contains project idea, description, user manual, full project view, and source code.

2. **CSE110 Project-RawFile.zip**  
   - Contains the source code files that can be imported into **NetBeans** to run the project.

---

## Step-by-Step Guide to Run the Project

### 1. Install NetBeans (if not installed)
- Download and install **NetBeans IDE** from [https://netbeans.apache.org/](https://netbeans.apache.org/)
- Ensure **Java JDK** is installed (JDK 8 or higher recommended).

### 2. Extract Project Files
- Extract `CSE110 Project-RawFile.zip` to a folder of your choice.

### 3. Open Project in NetBeans
1. Launch **NetBeans IDE**.
2. Go to **File â†’ Open Project**.
3. Navigate to the extracted folder and select the project.
4. Click **Open Project**.

### 4. Build the Project
- Right-click the project name in **Projects** panel.
- Click **Build** or **Clean and Build** to compile all files.

### 5. Run the Project
- Right-click the project name again.
- Select **Run**.
- The **Course Selection Management System** GUI will open.

## Features

- Students can **select courses**.
- Management can **add or remove courses** anytime.
- Easy-to-use **Java Swing interface** with **JFrame, JButton, JTable**.
- Demonstrates **OOP principles**: classes, objects, and encapsulation.
